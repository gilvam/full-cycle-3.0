
### docker compose

Run
``` shell
docker-compose up
```

Roda liberando o terminal como -d (detach)
``` shell
docker-compose up -d
```

Roda com novos builds
``` shell
docker-compose up -d --build
```

Lista apenas os containers do docker compose
``` shell
docker-compose ps
```

Deliga todos os containers
``` shell
docker-compose down
```
